# LLM Infrastructure & Deployment

Production-grade guidance for deploying LLMs on Kubernetes, cloud platforms, and edge devices.

## Skills in This Category

- **Kubernetes for LLMs** - GPU scheduling, StatefulSets, autoscaling
- **Container Optimization** - Docker, GPU support, multi-stage builds
- **Cloud Deployment** - AWS SageMaker, GCP Vertex AI, Azure ML
- **Serving Frameworks** - vLLM, SGLang, TGI, Triton, Ollama
- **Cost Optimization** - Spot instances, quantization, batch processing
- **Monitoring & Observability** - Prometheus, Grafana, OpenTelemetry

## Key Resources

- `llm-kubernetes-production.prompt.md` - Complete Kubernetes guide with manifests
- Research documents on infrastructure patterns
- Code examples for deployment automation
- Configuration templates for cloud platforms

## Getting Started

1. Choose your serving framework (vLLM recommended for most)
2. Set up Kubernetes cluster with GPU support
3. Deploy using provided manifests
4. Configure monitoring with Prometheus/Grafana
5. Implement cost optimization strategies

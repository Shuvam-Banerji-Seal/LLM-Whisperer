## File Naming Convention

All files follow a consistent naming pattern:

```
SKILL_CATEGORY_DOCUMENT_TYPE.md

Examples:
- GNN_COMPREHENSIVE_RESEARCH_INDEX.md
- GAT_IMPLEMENTATION_GUIDE.md
- FEDERATED_LEARNING_QUICK_REFERENCE.md
- TRANSFER_LEARNING_DOCUMENTATION_INDEX.md
```

**Document Types:**
- `COMPREHENSIVE_GUIDE` - Full technical reference
- `IMPLEMENTATION_GUIDE` - Production code + tutorials
- `QUICK_REFERENCE` - Cheat sheets + snippets
- `RESEARCH_SOURCES` - Bibliographies + citations
- `README` - Entry point + orientation
- `INDEX` - Navigation + learning paths
- `DOCUMENTATION_SUITE_OVERVIEW` - Package structure

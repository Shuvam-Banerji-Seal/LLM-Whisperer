# Configuration Module\n\n## Overview\n\nCentralized configuration management for all fine-tuning approaches. Provides easy import and access to all configuration classes.\n\n## Available Configurations\n\n- **BaseFinetuningConfig**: Foundation configuration for all approaches\n- **LoRAConfig**: Low-Rank Adaptation configuration\n- **QLoRAConfig**: Quantized LoRA configuration\n- **BehaviorTuningConfig**: Behavior-specific tuning\n- **AgenticTuningConfig**: Agentic system tuning\n- **MultimodalTuningConfig**: Vision-language and multimodal models\n- **RAGTuningConfig**: Retrieval-augmented generation\n- **RewardModelingConfig**: Reward model training\n\n## Usage\n\n```python\nfrom fine_tuning.configs import LoRAConfig, QLoRAConfig\n\n# Use any configuration\nconfig = LoRAConfig(\n    model_name=\"mistralai/Mistral-7B\",\n    r=16,\n    lora_alpha=32,\n)\n```\n\n## See Also\n\n- [Base Module](../base/README.md)\n- [LoRA Module](../lora/README.md)\n- [Templates Module](../templates/README.md)\n"

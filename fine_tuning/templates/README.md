# Templates Module\n\n## Overview\n\nPre-configured templates for common fine-tuning scenarios. Templates provide quick-start configurations for popular models and use cases.\n\n## LoRA Templates\n\n- **gpt2_lora_config()**: GPT-2 with LoRA (r=4)\n- **mistral_lora_config()**: Mistral-7B with LoRA (r=16)\n- **llama_lora_config()**: Llama-7B with LoRA (r=16)\n\n## QLoRA Templates\n\n- **mistral_qlora_config()**: Mistral-7B with QLoRA (r=16)\n- **llama_qlora_config()**: Llama-7B with QLoRA (r=16)\n- **llama_70b_qlora_config()**: Llama-70B with QLoRA (r=32)\n\n## Usage\n\n```python\nfrom fine_tuning.templates import mistral_lora_config, llama_qlora_config\n\n# Use template with default settings\nconfig = mistral_lora_config()\n\n# Override specific settings\nconfig = llama_qlora_config(\n    num_epochs=5,\n    batch_size=8,\n    learning_rate=1e-4,\n)\n\n# Create finetuner\nfinetuner = LoRAFinetuner(config)\n```\n\n## See Also\n\n- [Base Module](../base/README.md)\n- [LoRA Module](../lora/README.md)\n- [QLoRA Module](../qlora/README.md)\n"

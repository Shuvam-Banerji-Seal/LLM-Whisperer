# Task Prompt: Feature Delivery

## Objective

Deliver {{goal}} while preserving existing behavior outside accepted change scope.

## Required Inputs

- ticket or request
- acceptance_criteria
- impacted components
- risk constraints

## Execution Checklist

1. Build implementation plan with explicit test plan.
2. Identify unknowns and perform focused research if needed.
3. Implement minimal safe changes.
4. Run review and safety gate.
5. Produce release recommendation with rollback notes.

## Completion Criteria

- acceptance criteria met
- no unresolved critical/high review findings
- safety status is pass
